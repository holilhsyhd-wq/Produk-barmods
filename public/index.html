Cara Order

Via DANA + Gmail + bukti transfer (SS)

1. Pilih produk di bawah.
2. Tekan tombol Order via Gmail pada produk.
3. Sistem akan minta kamu bayar dulu via DANA ke nomor 083877842721.
4. Setelah bayar, isi data DANA & lanjut ke Gmail.
5. Di Gmail, wajib lampirkan screenshot (SS) bukti transfer sebelum kirim email.
